# Stablecoin Simulation Bot Dashboard

This Streamlit app simulates a portfolio of stablecoins (USDT, USDC, DAI) with minor low‑risk trades, live updates, and drift alerts. It is designed for continuous live updates every 5 minutes.

## Deployment

Deploy this project on Streamlit Community Cloud:

1. Sign in with your GitHub account.
2. Click **New App**.
3. Select this repository.
4. Click **Deploy**.

Enjoy!
