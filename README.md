# Live Stablecoin Simulation Bot

This Streamlit app simulates a virtual portfolio of stablecoins (USDT, USDC, DAI)  
with minor automated trades, live market prices via CoinGecko, and drift alerts.

## Deployment

1. Sign in to [Streamlit Community Cloud](https://share.streamlit.io/) with your GitHub account.
2. Create a new app using this repository.
3. Set the main file to `app.py`.
4. Click **Deploy**.
5. Open the live dashboard — it updates every 5 minutes automatically.
